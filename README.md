# 자연어 입력 문항 생성기

수능 영어 문항 AI 검색 시스템

## 프로젝트 개요

영어 교사가 자연어로 입력한 문항 생성 지침을 LLM이 분석하여 구조화된 검색 조건으로 변환하고, 구글 시트의 수능 문항 마스터 테이블에서 적합한 문항을 자동으로 검색·제공하는 시스템입니다.

## 주요 기능

- 자연어 입력을 통한 직관적인 문항 검색
- LLM 기반 검색 조건 자동 추출
- 구글 시트 연동을 통한 문항 데이터베이스 관리
- 검색 결과 미리보기 및 내보내기
- 검색 히스토리 및 즐겨찾기

## 기술 스택

- **Frontend**: React.js / Next.js, TypeScript, Tailwind CSS
- **Backend**: Node.js (Express) / Python (FastAPI)
- **LLM**: OpenAI GPT-4 / Anthropic Claude / Google Gemini
- **Database**: Google Sheets API, Firebase
- **Infrastructure**: Vercel / AWS / GCP
